## ⚔️ What is Kensei?

Kensei is a production-grade AutoML and MLOps platform designed to simplify
machine learning workflows for small teams and developers.

It enables users to:

• Upload datasets
• Automatically train multiple models
• Run hyperparameter optimization
• Track experiments and metrics
• Select the best model
• Deploy it instantly as a FastAPI endpoint

Kensei combines FastAPI, Celery, Optuna, and MLflow to provide a
lightweight, self-hosted alternative to cloud ML platforms like SageMaker or Vertex AI.
